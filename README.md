面试专用项目
餐饮管理平台 后端开发
时间：2025.05-2025.10
项目描述：一套面向 B 端商家的全链路餐饮管理系统，涵盖点餐、接单、配送及数据报表功能。
核心技术：Spring Boot、Redis、WebSocket、Nginx，JWT, Spring Cache
实时交互优化：摒弃传统轮询机制，集成 WebSocket 实现服务端向商家的毫秒级消息推送（来单 / 催单），显著降低服务资源消耗。
高性能缓存架构：针对高频访问的店铺菜品数据，设计 Spring Cache + Redis 多级缓存方案，大幅降低数据库 I/O 压力，接口响应速度提升。
接口安全与网关：采用 Nginx 作为反向代理服务器进行负载均衡；基于JWT实现登录鉴权和刷新登录状态，保障数据传输安全。
